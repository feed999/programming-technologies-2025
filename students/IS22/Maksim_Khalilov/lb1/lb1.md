## Лабораторная работа №1. Знакомство с OpenAI API. Написание простого текстового ассистента

- Цель: научиться работать с OpenAI API, чтобы создать простого текстового ассистента, который может отвечать на ваши вопросы.

## 1. Настройка окружения

- Настройка виртуального окружения - python -m venv venv
- venv\Scripts\activate
- Установка библиотек - pip install openai
- Создадим файл - requirements.txt
- Сохронение настроек - pip freeze > requirements.txt
- Получаем - OPENAI_API_KEY
